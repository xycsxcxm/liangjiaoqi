相机无法加载

请查看下方解决方案解决此问题

相机问题诊断
安全上下文问题
浏览器要求通过HTTPS或localhost访问相机API。当前运行在：file://
权限被拒绝
用户未授予相机访问权限或浏览器阻止了权限请求
设备兼容性问题
您的设备或浏览器可能不支持所需API，或没有可用摄像头
API支持检测
您的浏览器支持：getUserMedia API

解决方案：

  1. 使用HTTPS协议：将应用部署在HTTPS服务器上，或使用localhost开发环境
  2. 授予相机权限：检查浏览器地址栏中的相机权限图标并允许访问
  3. 检查物理设备：确保设备有可用的摄像头且未被其他应用占用
  4. 更新浏览器：使用最新版本的Chrome、Firefox或Edge浏览器
  5. 检查安全设置：禁用可能阻止相机访问的浏览器扩展或防火墙
  6. 设备兼容性：确保设备具有可用的摄像头并支持媒体API

应用使用指南

  1. 解决相机问题后，点击"开始测量"按钮
  2. 在屏幕上依次标记三个点：顶点 → 第一条边端点 → 第二条边端点
  3. 应用将自动计算并显示角度值
  4. 使用"清除标记"按钮重置测量
  5. 使用"翻转相机"按钮在前置和后置摄像头间切换
